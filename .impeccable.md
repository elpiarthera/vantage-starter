# VantageStarter — Impeccable Design Context

## Identity

VantageStarter is a production-ready Next.js + Convex + Clerk SaaS starter kit.
Target buyer: developers building B2B SaaS products who want a confident, non-generic foundation.

## Font Stack (locked)

- **Display + Body:** Instrument Sans — humanist, editorial, distinctive
- **Code:** Geist Mono
- **Banned:** Inter, Plus Jakarta Sans, DM Sans, Outfit

## Color System (locked)

- **Primary hue:** 44° (warm amber-gold) — `oklch(0.62 0.16 44)` light / `oklch(0.72 0.16 44)` dark
- **Color space:** OKLCH throughout — never HSL, never hex except for third-party API constraints
- **Banned hues:** SaaS blue (220–240°), AI purple (270–290°), Vercel green

## Typography Rules

- Display headings: `letter-spacing: -0.03em`
- Body: `letter-spacing: -0.01em`
- No all-caps headings
- No font-weight > 700 in body copy

## Border Radius Contract

| Element | Value |
|---------|-------|
| Cards, panels, dialogs | `rounded-xl` (12px) |
| Buttons | `rounded-lg` (8px) |
| Badges, tags | `rounded-full` |

## Spacing Contract

| Context | Value |
|---------|-------|
| Section vertical padding | `py-20 md:py-32` |
| Content max-width | `max-w-5xl mx-auto px-6` |
| Grid gutter | `gap-8 md:gap-12` |
| Inline element gap | `gap-3` |
| Component internal padding | `p-6` (cards), `px-4 py-2` (buttons) |

## Anti-patterns (banned)

- Glassmorphism (`backdrop-blur` + semi-transparent backgrounds in auth UI)
- Purple gradients
- Abstract shape backgrounds (blobs, rings, noise)
- Card-in-card nesting
- Dashboard screenshots in the hero
- Bounce or elastic easing
- Animations over 500ms inside the app
- Framer Motion inside authenticated views (landing Hero only)
- Emojis in UI copy
- Exclamation marks in empty states

## Shadow Scale

Use VantageCRM's Tailwind shadow scale. Tailwind defaults are banned — visually inconsistent.

## Hero Treatment

Typographic-only. Diagonal gradient sweep (`bg-gradient-to-br`), OKLCH lightness sweep only.
Light: `0.98 → 0.93`. Dark: `0.08 → 0.14`. No warm-to-cool crossfade.

## Iconography

Lucide React exclusively. Default 16px (`size-4`). Empty state exception: 48px (`size-12`) in `text-muted-foreground`.
