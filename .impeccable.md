## Utility Classes (from globals.css + LitUI)

### `.card-elevated`
Shadow depth with hover lift. Use only when content is truly distinct and actionable.
```css
box-shadow:
  0 4px 6px -1px oklch(0 0 0 / 0.05),
  0 10px 15px -3px oklch(0 0 0 / 0.08),
  0 20px 25px -5px oklch(0 0 0 / 0.06);
transition: box-shadow 0.4s ease-out-expo, transform 0.4s ease-out-expo;
```
Hover: lifts 2px, shadows deepen. Dark mode: higher alpha.

### `.btn-shadow`
Subtle depth for primary CTAs.
```css
box-shadow:
  0 1px 2px 0 oklch(0 0 0 / 0.05),
  inset 0 1px 0 oklch(1 0 0 / 0.1);
```

### `.icon-container`
Feature grid icon wrapper. 40px square, rounded-lg, muted background.
```css
@apply size-10 rounded-lg bg-muted/50 flex items-center justify-center;
```
Never place a large icon above a heading â€” use inline with text instead.

### `.active-scale`
Press feedback for CTAs.
```css
:active { transform: scale(0.96); transition: transform 100ms ease-out-expo; }
```

### Entrance animations

### Reveal on scroll

## Anti-patterns (banned)

- Blue glow effects
- Glassmorphism / backdrop-blur decoration
- Purple-to-blue gradients
- Gradient text
- Cards inside cards
- Large icons above headings
- Bounce or elastic easing
- Animations over 500ms
- Dashboard screenshots in hero
- Generic SaaS illustration
- Any visual that screams "template"

## Motion

- 150ms ease-out-expo for micro-interactions
- 300ms ease-out-expo for state changes
- Only animate transform and opacity
- Always respect prefers-reduced-motion
