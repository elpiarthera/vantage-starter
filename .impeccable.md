# VantageStarter — Impeccable Design Context

## Target audience
Developers and technical founders building SaaS products. They judge quality in the first 3 seconds. If the landing page looks like a template, they leave.

## Use cases
Landing page for a Next.js + Convex + Clerk boilerplate. Must convince a developer to buy in under 10 seconds. The page IS the product demo.

## Brand personality
Editorial. Confident. Premium. Think Linear, Vercel, Stripe — not generic SaaS template. Bold typography, restrained color, generous whitespace.

## Design reference
ElevenLabs Hacks (hacks.elevenlabs.io) — sharp edges, custom typography, restrained palette, editorial confidence.

## Font Stack

- **Headings:** Space Grotesk 600/700 — geometric, distinctive
- **Body:** Inter 400/500
- **Code:** Geist Mono
- **Banned:** Instrument Sans, Plus Jakarta Sans, DM Sans, Outfit, Roboto, Open Sans

## Color System (OKLCH)

- Background: oklch(0.13 0.01 240) — near-black, tinted blue
- Surface: oklch(0.17 0.01 240) — elevated
- Border: oklch(0.25 0.01 240) — subtle
- Text primary: oklch(0.93 0.01 240) — warm white
- Text secondary: oklch(0.65 0.01 240) — muted
- Primary: oklch(0.62 0.18 240) — desaturated blue (used sparingly)
- Accent: oklch(0.75 0.14 65) — warm gold/amber (for highlights only)
- Danger: oklch(0.65 0.2 25)

## Shape Language

| Element | Border radius |
|---------|--------------|
| Cards, panels, dialogs | 0px (sharp, editorial) |
| Buttons | 999px (pill) |
| Inputs | 6px |
| Badges, tags | 999px |

## Spacing

| Context | Value |
|---------|-------|
| Section vertical padding | 96px (py-24) |
| Content max-width | max-w-6xl mx-auto px-6 lg:px-12 |
| Grid gutter | gap-6 lg:gap-8 |
| Card internal padding | p-8 |
| Button padding | px-6 py-3 |

## Anti-patterns (banned)

- Blue glow effects
- Glassmorphism / backdrop-blur decoration
- Purple-to-blue gradients
- Gradient text
- Cards inside cards
- Large icons above headings
- Bounce or elastic easing
- Animations over 500ms
- Dashboard screenshots in hero
- Generic SaaS illustration
- Any visual that screams "template"

## Motion

- 150ms ease-out for micro-interactions
- 300ms ease-out for state changes
- Only animate transform and opacity
- Always respect prefers-reduced-motion

## Quality bar

If someone looks at this and thinks "AI made this" or "this is a template" — it fails. The design must have a point of view.
